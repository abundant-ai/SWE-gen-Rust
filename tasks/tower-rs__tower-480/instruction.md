`tower::buffer` has a regression after switching to a semaphore-based bounded channel implementation: tasks that are blocked in `Service::poll_ready` waiting for buffer capacity can become permanently stuck if the buffer worker terminates (for example, because the inner service fails) before capacity becomes available.

Reproduction scenario: create a `tower::buffer::Buffer` with a small capacity and drive it until it is at/over capacity so that subsequent callers observe `Poll::Pending` from `poll_ready` while waiting for capacity/reservation. If, while callers are still pending in `poll_ready`, the buffer’s worker task terminates (e.g., the inner service returns an error or the worker shuts down), those pending callers are never woken. This effectively leaks those tasks: they will never be polled to readiness again and will hang indefinitely.

Expected behavior: when the buffer can no longer make progress because its worker has terminated (due to shutdown or inner-service failure), any tasks currently waiting for capacity in `poll_ready` must be woken so they can observe termination and return promptly (typically `Poll::Ready(Err(_))` from `poll_ready`, or otherwise transition out of the pending state in a way that does not hang). In other words, `poll_ready` must not remain pending forever once the buffer is closed/failed.

Actual behavior: when at capacity, `poll_ready` can remain pending forever after the worker terminates, because the semaphore used to model capacity does not automatically wake all waiters on close the way a bounded MPSC sender would on receiver drop.

Fix required: ensure that dropping/terminating the buffer worker (including inner-service failure paths) triggers wakeups for all tasks blocked waiting for capacity/reservations, so that `Buffer` does not hang/leak tasks in this state. The behavior should be consistent whether termination happens via normal shutdown or via an error from the inner service. The wakeup mechanism must cover all waiters that were pending on capacity at the time of termination.
